# 🧠 Bullying Prediction App (Streamlit)

## 🇮🇩 Deskripsi
Aplikasi ini merupakan sistem prediksi risiko bullying di sekolah berbasis web menggunakan Python dan Streamlit.  
Pengguna dapat mengisi data, lalu sistem akan memprediksi apakah seseorang berisiko mengalami bullying atau tidak.

### ✨ Fitur
- Input data siswa
- Prediksi risiko bullying
- Tampilan interaktif berbasis web

---

## 🇬🇧 Description
This is a web-based bullying risk prediction application built using Python and Streamlit.  
Users can input data and the system will predict whether someone is at risk of bullying.

### ✨ Features
- User input form
- Bullying risk prediction
- Interactive web interface

---

## 🛠️ Technologies
- Python
- Streamlit
- Scikit-learn
- Pandas
